# Car Price Prediction using Machine Learning

## Overview

This project predicts the selling price of used cars using Machine Learning. The model is trained on historical car sales data and estimates the selling price based on various input features.

## Objectives

* Predict the selling price of a used car.
* Perform data preprocessing and visualization.
* Train and evaluate a Machine Learning model.
* Build a simple prediction system.

## Features

* Data Cleaning
* Data Visualization
* Feature Engineering
* Linear Regression Model
* Price Prediction

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Joblib

## Dataset Features

* Car Name
* Year
* Selling Price
* Present Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission
* Owner

## Machine Learning Algorithm

Linear Regression

## Project Workflow

1. Collect Dataset
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Price Prediction

* Improve model accuracy using advanced algorithms.
